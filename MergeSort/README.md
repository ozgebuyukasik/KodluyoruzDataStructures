# Merge Sort

## a) Steps of the sorting algorithm for [16,21,11,8,12,22]

1. [16, 21, 11] - [8, 12, 22]
    
    a. [16] - [21, 11],

        i. [21]

        ii. [11]

        iii. [11, 21]
    
        iv.[11, 16, 21]
    b. [8] - [12, 22]

        i. [12] - [22]

        ii. [12, 22]

        iii. [8, 12, 22]
    c. [8, 11, 12, 16, 21, 22]

## b) Big-O

O(n*log n)
